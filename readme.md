# Flycat Batch OCR
## Overview
Flycat Batch OCR is a lightweight, fast and offline batch OCR desktop app designed for Windows. It allows you to extract text in bulk from images, screenshots and scanned files without uploading any data to the cloud.

All recognition runs locally on your device, ensuring **100% privacy protection** and no network dependency.

## Key Features
- **Batch OCR Processing**
  Import multiple images at once and extract text in one click.
- **Fully Offline Recognition**
  No internet required, all OCR computation runs locally.
- **Multi-Format Support**
  Support PNG, JPG, JPEG, BMP and common image formats.
- **Quick Copy & Export**
  Copy recognized text directly to clipboard or save as TXT file.
- **Simple & Clean UI**
  User-friendly interface, no complicated settings, ready to use out of the box.
- **Lightweight & Low Resource Usage**
  Small installation package, low CPU and memory consumption.

## How to Use
1. Launch Flycat Batch OCR
2. Drag and drop images or click to import image files
3. Start one-click batch text recognition
4. Preview the result, copy text or export to TXT

## Privacy Policy
All image files and text recognition data are processed **only locally** on your computer.
No user data, images or recognized content will be uploaded, collected or shared to any server.

## System Requirements
- OS: Windows 10 / Windows 11 (64-bit)
- Minimum RAM: 4GB
- Storage: Less than 200MB installation space

## Download & Update
Available exclusively on **Microsoft Store**.
Auto-update supported to keep you with the latest features and performance optimizations.
